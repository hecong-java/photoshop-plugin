## Interaction Specifications

### 1. Right-Click Context Menu

**Trigger:** User right-clicks on any `<img>` element with `data-prompt-reverse` attribute.

**Behavior:**
- Prevent default browser context menu
- Show single menu item: "反推提示词"
- Menu positioned at cursor coordinates, clamped to viewport
- Menu dismisses on: click outside, Escape key, scroll, or item click
- Menu dismisses if another context menu opens

**Visual:**
- Menu container: `background: #1a1f2e`, `border: 1px solid rgba(255,255,255,0.15)`, `border-radius: 8px`, `padding: 4px 0`
- Menu item: `padding: 8px 16px`, `font-size: 13px`, `color: #e2e8f0`, hover `background: rgba(66,153,225,0.15)`
- z-index: 1500 (above all existing overlays: viewer=999-1200, dialogs=1400)

### 2. PromptReverseFlow Modal (Multi-Step)

**Step 1: Image Preview ("确认图片")**
- Show clicked image as preview (max 300px height, object-fit contain)
- Image source extracted from right-clicked `<img>` element
- "取消" button (secondary) and "下一步" button (primary, #4299e1)

**Step 2: Template Selection ("选择描述风格")**
- Show 4 template cards in 2x2 grid
- Each card: template name (15px, semibold) + description (12px, secondary)
- Cards use existing .workflow-toolkit-item pattern from Draw.css
- Selected card: border highlight #38bdf8, background rgba(56,189,248,0.12)
- "返回" button (secondary) and "开始分析" button (primary)
- "开始分析" disabled until a template is selected

**Step 3: Loading ("正在分析图片...")**
- Show spinner (reuse .spinner from Draw.css, 16px circle)
- Show progress text: "正在分析图片，请稍候..."
- Disable all buttons; show "取消" button (secondary) that aborts request
- Timeout: 60 seconds

**Step 4: Result ("分析结果")**
- Image thumbnail (64px, top-left corner)
- Selected template name label (12px, muted)
- Result text area (scrollable, min-height 120px, max-height 300px)
  - `font-size: 14px`, `line-height: 1.6`, `color: #e2e8f0`
  - `background: rgba(15,23,42,0.7)`, `border: 1px solid rgba(255,255,255,0.08)`, `border-radius: 8px`
  - `padding: 12px`, read-only (user selects text to copy manually, or uses buttons)
- Action buttons row:
  - "复制到剪贴板" (primary, #4299e1)
  - "填入提示词" (info, rgba(6,182,212,0.22) -- only visible on Draw page)
  - "重新分析" (secondary) -- returns to Step 2
- Success feedback: button text changes to "已复制" for 2 seconds after copy

**Modal Container:**
- Overlay: `position: fixed, inset: 0, background: rgba(2,6,23,0.7), z-index: 1500`
- Modal: `min-width: 360px, max-width: 480px, background: #0b1220, border-radius: 14px, padding: 24px`
- Header: title (15px, 600, #f8fafc), close button (top-right)
- Close behavior: overlay click, Escape key, close button

### 3. Settings Page: DashScope Config Card

**Position:** New card in settings-grid, below ComfyUI Connection card (grid is 3-column at >=1120px)

**Content:**
- Card header: "提示词反推" + status badge ("已配置" green / "未配置" red)
- API Key input: type="password", placeholder="sk-...", follows existing .text-input style
- Model selector: `<select>` with options: Qwen VL Max, Qwen VL Plus, Qwen3 VL Plus
- Default model: qwen-vl-plus
- Follows existing .settings-card and .form-group patterns exactly

### 4. Image Data Attributes

Images that support right-click reverse:
- Draw page: `.preview-image` (generated output), `.preview-strip-item img` (output thumbnails), `.image-preview` (uploaded input images)
- History page: `.history-item-thumbnail img` (thumbnails), `.history-viewer-body img` (viewer)

Attribute to add: `data-prompt-reverse`

## State Machine

```
CLOSED -> (right-click "反推提示词") -> PREVIEW
PREVIEW -> (click "下一步") -> TEMPLATE
PREVIEW -> (click "取消" / overlay / Escape) -> CLOSED
TEMPLATE -> (click "开始分析") -> LOADING
TEMPLATE -> (click "返回") -> PREVIEW
TEMPLATE -> (click "取消" / overlay / Escape) -> CLOSED
LOADING -> (API success) -> RESULT
LOADING -> (API error) -> RESULT (with error displayed)
LOADING -> (click "取消" / abort) -> CLOSED
RESULT -> (click "重新分析") -> TEMPLATE
RESULT -> (click "取消" / overlay / Escape) -> CLOSED
```

On any new right-click while modal is open: abort current request, reset state, start fresh from PREVIEW.
