    1. Export interface `ClusterPresetMeta`:
       ```
       { id: string; template_id: string; name: string; parameters: Record<string, unknown>; scope: 'personal' | 'shared'; owner_id: string | null; created_at: string; updated_at: string; }
       ```

    2. Export function `listPresets(templateId: string): Promise<ClusterPresetMeta[]>`:
       - Call `await ensureValidToken()` first.
       - Get serverUrl from `useLemonGridStore.getState().serverUrl`, strip trailing slashes.
       - Call `lemongridFetch(\`${serverUrl}/api/v1/templates/${templateId}/presets?page_size=100\`)`.
       - If response not ok, throw `Error(\`List presets failed: \${response.status}\`)`.
       - Parse JSON as `{ items: ClusterPresetMeta[]; total: number }`, return `data.items || []`.

    3. Export function `createPreset(templateId: string, name: string, parameters: Record<string, unknown>, scope?: 'personal' | 'shared'): Promise<ClusterPresetMeta>`:
       - Default scope to `'personal'`.
       - Call `await ensureValidToken()`.
       - POST to `${serverUrl}/api/v1/templates/${templateId}/presets` with body `{ template_id: templateId, name, parameters, scope }`.
       - If response status is 409, throw `Error('PRESET_NAME_CONFLICT')`.
       - If not ok, throw `Error(\`Create preset failed: \${response.status}\`)`.
       - Return parsed JSON as ClusterPresetMeta.

    4. Export function `updatePreset(templateId: string, presetId: string, data: { name?: string; parameters?: Record<string, unknown>; scope?: string }): Promise<ClusterPresetMeta>`:
       - Call `await ensureValidToken()`.
       - PUT to `${serverUrl}/api/v1/templates/${templateId}/presets/${presetId}` with body.
       - If not ok, throw `Error(\`Update preset failed: \${response.status}\`)`.
       - Return parsed JSON.

    5. Export function `deletePreset(templateId: string, presetId: string): Promise<void>`:
       - Call `await ensureValidToken()`.
       - DELETE to `${serverUrl}/api/v1/templates/${templateId}/presets/${presetId}`.
       - If not ok AND status not 204, throw error.
