  <what-built>
    Complete workflow parameter preset management feature including:
    - Preset toolbar with dropdown, add(+), settings(gear), import, export buttons
    - Preset save/load/switch with parameter value application
    - Delete confirmation dialog
    - Unsaved changes warning on preset switch
    - Import with conflict resolution dialog
    - Export preset to JSON file
    - Image reference invalidation warning
  </what-built>
  <how-to-verify>
    **Pre-requisite:** Open the plugin in Photoshop with ComfyUI running.

    **Test 1 - Preset Toolbar Display:**
    1. Select a workflow from the workflow picker
    2. Verify the preset toolbar appears at the top of the parameter section (above form fields)
    3. Verify toolbar shows: dropdown with "选择预设...", + button, gear button, import button, export button
    4. Without a workflow selected, verify toolbar is disabled (grayed out)

    **Test 2 - Save Preset:**
    1. Modify some parameter values (change text, sliders, etc.)
    2. Click the + button
    3. Verify a new preset named "预设 1" appears in the dropdown and is selected
    4. Click + again, verify "预设 2" appears

    **Test 3 - Switch Presets:**
    1. Modify some parameters
    2. Click + to save as "预设 3"
    3. Select "预设 1" from dropdown
    4. Verify parameter values change to those saved in "预设 1"
    5. Select "预设 3" again, verify values change back

    **Test 4 - Unsaved Changes Warning (D-08):**
    1. Select "预设 1"
    2. Change a parameter value (don't save)
    3. Try to switch to "预设 2"
    4. Verify confirmation dialog appears with "当前参数有未保存的修改，切换预设将丢失这些修改。"
    5. Click "不保存并切换" -- verify switch happens
    6. Click "取消" -- verify you stay on current preset

    **Test 5 - Delete Preset (D-07):**
    1. Click the gear/settings button
    2. Find and click the delete button for a preset
    3. Verify confirmation dialog: "确定要删除预设「{name}」吗？此操作不可撤销。"
    4. Click "删除" -- verify preset is removed from dropdown
    5. Repeat but click "取消" -- verify preset still exists

    **Test 6 - Rename Preset:**
    1. Click the gear/settings button
    2. Click on a preset name to enter edit mode
    3. Type a new name and press Enter
    4. Verify the name updates in the dropdown
    5. Press Escape -- verify rename is cancelled

    **Test 7 - Export Preset (D-12):**
    1. Select a preset from the dropdown
    2. Click the export button
    3. Verify a JSON file is created in the presets folder
    4. Verify file contains inputValues and imageFilenames

    **Test 8 - Import Preset with Conflict (D-14):**
    1. Click the import button
    2. Select a preset file with a name that already exists
    3. Verify conflict dialog appears with 3 options: "覆盖", "跳过", "重命名"
    4. Test each option

    **Test 9 - Image Reference Validation (D-11):**
    1. Create a preset with an image input that has a filename reference
    2. Manually delete or rename the image file on ComfyUI side
    3. Apply the preset
    4. Verify the image input shows "图片引用已失效，请重新上传" warning
    5. Upload a new image -- verify warning disappears
  </how-to-verify>
