    **Implementation details:**

    1. Import config service:
    ```typescript
    import { loadPluginConfig, type PluginConfig } from '../services/config';
    ```

    2. shouldDisplayNode logic:
    - If config is null or nodes array is empty, return true (show all)
    - Return true if any node.class_type matches classType
    - Return false otherwise

    3. getAllowedInputs logic:
    - If config is null or nodes array is empty, return null (all inputs)
    - Find node with matching class_type
    - If not found, return null (all inputs)
    - If found and has inputs array, return it
    - If found but no inputs property, return null (all inputs)

    4. loadConfig action:
    - Set isLoading=true, error=null
    - Call loadPluginConfig() from service
    - Set config, isLoading=false, loadedAt timestamp
    - On error: set error message, isLoading=false, fallback to DEFAULT_CONFIG

    5. Use persist middleware:
    - name: 'Ningleai-config'
    - partialize: only config and loadedAt (not isLoading/error)
